Async / Await

- async함수앞 키워드로 붙입니다 => 함수가 promise를 리턴
- await => 를 사용하면 함수가 실행을 일시중지하고 
promise가 해결될때 까지 기다린후 계속진행

Promises

- 결과를 약속 드립니다 
0) Promise Object 속성
- Pending(보류) : 정의되지 않음(undefined)
- fulfilled(성취) : 결과값(a result value)
- rejected(거부) : 오류객체(an error object)

1) 코드생성 [Producing code] : 시간이 걸릴수 있는 코드입니다
생성코드가 결과를 얻으려면 아래에 두콜백중에 하나를 호출해야 합니다
Success : myResolve(result value)
Error : myReject(error object)

2) 소비코드 [Consuming code] : 결과를 기다려야 하는 코드
자바스크립트, 자바, 리액트, 뷰 는 객체지향 언어입니다
Promise는 생성 코드와 소비코드를 연결하는 객체입니다

async 

- 다른 함수와 병렬로 실행되는 함수를 비동기라고 합니다
좋은예는 javascript setTimeout()입니다
- 병렬 : 여러개의 일이 동시에 이루어지거나 여로요소가 나란히 놓여 있다는 뜻으로 사용됩니다
- 실생활 => "도로가 병렬로 나 있다 " -> 도로가 나란히 놓여 있음
- 컴퓨터 / 전자 공학에서의 병렬 => 여러작업이나 처리를 동시에 수행하는 방식
- 자바스크립트에서는 비동기적으로 작업을 처리할수 있어서 여러작업이 병렬로 진행되는 것처럼
실행됩니다
